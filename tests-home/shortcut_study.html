Resumo HTML - Características Importantes

1. Estrutura Básica

<!DOCTYPE html>
<html lang="pt-BR">
  <head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
  </head>
  <body>
    Conteúdo da página
  </body>
</html>

2. Elementos sem Fechamento (void elements)

  • <br> - quebra de linha
  • <img> - imagem
  • <input> - campo de formulário
  • <hr> - linha horizontal
  • <meta> - metadados
  • <link> - ligação com arquivos externos

3. Elementos Semânticos

  • <header> - cabeçalho da página ou seção
  • <nav> - navegação
  • <main> - conteúdo principal
  • <article> - conteúdo independente
  • <section> - seção de conteúdo
  • <aside> - conteúdo complementar
  • <footer> - rodapé

4. Formulários

<form action="/enviar" method="post">
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" required>

  <label for="senha">Senha:</label>
  <input type="password" id="senha" name="senha" required>

  <button type="submit">Entrar</button>
</form>

5. Acessibilidade

  • alt em <img> - descrição alternativa
  • aria-label, aria-hidden - atributos ARIA para leitores de tela
  • label + for - associar rótulo ao input
  • tabindex - controle de navegação com Tab

6. Comentários

<!-- Este é um comentário HTML -->

7. Incorporar Conteúdo

  • <iframe src="site.html"> - incorpora outra página
  • <video> e <audio> - mídia com suporte nativo
  • <script src="script.js"></script> - JavaScript externo
  • <link rel="stylesheet" href="style.css"> - CSS externo

8. Outras Tags Úteis

  • <strong> e <em> - ênfase semântica
  • <span> e <div> - contêineres genéricos
  • <ul>, <ol>, <li> - listas
  • <table>, <tr>, <td>, <th> - tabelas
  • <details> e <summary> - conteúdo expansível
